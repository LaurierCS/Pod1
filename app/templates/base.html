Welcome

The goal of this program is to sort through a vast number of comments from a tweet and examine what the opinions, appraisals, emotions, or attitudes towards a topic, person or entity is.

Instructions

1. Find the twitter post in question and copy its link in the URL (on mobile, click ‘share’ and copy the link)
2. Paste the link into this website and click ‘GO’ to analyze the sentiment of the post

    About This Project

    Sentiment analysis is a machine learning tool that analyzes texts for polarity, from positive to negative. By training machine learning tools with examples of emotions in text, machines automatically learn how to detect sentiment without human input. From a high-level, our goal is to create a functional web app that can be used by anyone to evaluate sentiments of specific things from websites or social media sites.
